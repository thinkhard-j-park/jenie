# Jenie Spring Data MongoDB

## 다중 몽고 클러스터 접근 설정
### 몽고 클러스터 DBConn 데이터베이스 및 컬렉션
- 각 클러스터에는 링크와 같은 형태의 여러 데이터베이스 접근을 위한 키와 데이터베이스 연결 정보를 담고 있는 컬렉션을 구성한다.
- 어플리케이션에서 접근하는 여러 클러스터간에서 키는 중복되지 않은 유일한 값이어야 한다.

### 어플리케이션 설정
